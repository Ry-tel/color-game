Hello user! This website is a simple color pickup game. On the top of the screen you will see the RGB color written in 8-bit per channel notation, i.e.: (255, 233, 0). Your goal is to guess which color is presented by those numbers. Please follow the Link to see how the RGB color mixing works. Site was build with vanilla JavaScript, Bootstrap is added only for displaying this message.

Have fun playing the game!
